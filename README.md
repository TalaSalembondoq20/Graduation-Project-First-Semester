# Waste Tyre Pyrolysis Simulation and Feasibility Evaluation

This repository contains the Markdown version of the graduation project report: **Simulation, Economic, Environmental, and Digital Monitoring Evaluation of a Small-Scale Waste Tyre Pyrolysis System**.

## Repository Structure

```text
docs/   Markdown sections of the report
media/  Extracted figures and images used by the Markdown files
```

## Main Sections

1. Introduction
2. Literature Review
3. Research Methodology
4. Simulation Model Development
5. Simulation Results and Analysis
6. Economic Feasibility Analysis
7. Environmental Impact Analysis
8. Proposed BioFuel Mobile Application Interface
9. Discussion
10. Conclusion and Recommendations
11. References
12. Appendices

## Tools and Methods

- Simio discrete-event simulation
- Economic feasibility analysis
- Python sensitivity analysis
- Environmental impact assessment
- Digital monitoring interface prototype

## Academic Use

This project is prepared for academic purposes as part of the Graduation Project course.
