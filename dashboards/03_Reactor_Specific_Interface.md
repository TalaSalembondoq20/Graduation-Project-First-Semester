# Reactor-Specific Interface

## Description

The reactor-specific interface focuses on the pyrolysis reactor because it is the main bottleneck in the system. The reactor operates in batch mode and has a long processing cycle compared with preprocessing stages.

## Main Reactor Indicators

The reactor dashboard may include:

- Reactor status
- Current batch size
- Batch start time
- Estimated completion time
- Temperature indicator
- Material ratio
- Output distribution
- Maintenance alerts
- Downtime warnings

## Importance

The reactor interface is important because any delay or downtime in the reactor directly affects system throughput, buffer accumulation, waiting time, and financial performance.
