# Mobile Monitoring Dashboard

## Overview

The proposed dashboard is a mobile-based monitoring prototype for the waste tyre pyrolysis plant. It is designed to display operational information in a clear and practical way for operators, engineers, and managers.

The dashboard supports the integrated evaluation of the system by linking the simulation model, economic feasibility results, environmental assessment, and operational monitoring needs.

## Dashboard Objectives

The main objectives of the dashboard are:

1. Monitor the overall condition of the plant.
2. Track the pyrolysis reactor as the main bottleneck.
3. Display material flow and batch status.
4. Show output distribution such as oil, carbon black, gas, light fuel fraction, and steel wires.
5. Provide alerts for operational issues.
6. Support decision-making through simple visual indicators.

## Dashboard Value

The dashboard adds a digital monitoring layer to the project. It helps explain how the plant can be monitored in practice, especially because the reactor has long processing time and strongly affects throughput, waiting time, and production capacity.
