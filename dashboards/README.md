# Dashboards

This folder documents the digital monitoring dashboard component of the Small-Scale Waste Tyre Pyrolysis System project.

## Purpose

The dashboard section represents the proposed BioFuel mobile monitoring interface used to support plant-level and reactor-level monitoring. The dashboard is not a full industrial control system; it is a prototype interface that explains how operators and managers can monitor the pyrolysis process visually.

## Included Markdown Files

- `01_Mobile_Monitoring_Dashboard.md`: Overview of the monitoring dashboard concept.
- `02_General_Dashboard_Interface.md`: Description of the general plant dashboard.
- `03_Reactor_Specific_Interface.md`: Description of the reactor-focused dashboard.
- `04_Reports_and_Alerts.md`: Description of reports, alerts, and operational notifications.
