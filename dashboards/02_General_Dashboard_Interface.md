# General Dashboard Interface

## Description

The general dashboard interface provides a plant-level view of the waste tyre pyrolysis system. It is designed to summarize the most important operational indicators in one screen.

## Main Monitoring Elements

The general dashboard may include:

- Plant operating status
- Daily tyre input
- Batch progress
- Buffer level
- Reactor operating condition
- Machine status
- Output quantities
- Maintenance status
- Alerts and warnings

## Operational Role

This interface allows the user to quickly understand whether the plant is operating normally or whether a critical process requires attention. It is especially useful for monitoring the connection between preprocessing stages and the reactor.
