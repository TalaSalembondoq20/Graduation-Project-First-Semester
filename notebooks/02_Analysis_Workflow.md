# Analysis Workflow

## Step 1: Define Inputs

The first step is to define the main input assumptions, including CAPEX, OPEX, daily product profit, operating days, and production capacity.

## Step 2: Calculate Baseline Feasibility

The baseline scenario calculates expected daily, monthly, and annual profit.

## Step 3: Calculate Payback and ROI

The model then calculates payback period and return on investment to evaluate financial attractiveness.

## Step 4: Run Sensitivity Analysis

Sensitivity analysis tests how the project performs under cost increase and profit decrease scenarios.

## Step 5: Visualize Results

Charts can be generated to compare different scenarios and identify the most critical financial risks.

## Step 6: Interpret Results

The final step is to interpret whether the project remains feasible under different assumptions and to identify the variables that require management attention.
