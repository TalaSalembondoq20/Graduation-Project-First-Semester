# Notebooks

This folder documents the notebook workflow for the Small-Scale Waste Tyre Pyrolysis System project.

## Current Status

The project documentation is provided in Markdown format. If a Jupyter Notebook or Google Colab notebook is added later, it should be placed in this folder.

## Included Markdown Files

- `01_Notebook_Overview.md`: Explains the purpose of notebooks in the project.
- `02_Analysis_Workflow.md`: Explains the suggested analysis workflow.

## Suggested Notebook Name

If a notebook is added later, use the following name:

`pyrolysis_financial_sensitivity_analysis.ipynb`
