# Models

This folder documents the main analytical models used in the Small-Scale Waste Tyre Pyrolysis System project.

## Included Markdown Files

- `01_Simulation_Model_Description.md`: Simio discrete-event simulation model.
- `02_Economic_Feasibility_Model.md`: CAPEX, OPEX, profit, payback, and ROI model.
- `03_Financial_Sensitivity_Model.md`: Python-based financial sensitivity analysis.
- `04_Environmental_Assessment_Model.md`: Environmental and sustainability assessment model.

## Purpose

The models section explains how the project moves from process understanding to technical, economic, and environmental evaluation.
