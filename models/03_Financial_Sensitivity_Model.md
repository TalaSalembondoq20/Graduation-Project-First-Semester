# Financial Sensitivity Model

## Purpose

The financial sensitivity model evaluates how project profitability changes when costs increase or product profits decrease.

## Tool Used

Python is used to calculate and compare different sensitivity scenarios.

## Sensitivity Variables

The model can test the impact of:

- Increase in operating costs
- Decrease in product profit
- Variation in product price
- Reactor downtime
- Changes in daily production capacity

## Main Outputs

The sensitivity analysis produces:

- Scenario-based monthly net profit
- Scenario-based annual net profit
- Profit risk comparison
- Feasibility status under different assumptions

## Importance

Sensitivity analysis is important because pyrolysis feasibility depends strongly on product prices, operating costs, and reactor reliability.
